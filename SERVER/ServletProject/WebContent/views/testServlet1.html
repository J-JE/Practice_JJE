GET 방식으로 페이지 요청해보기

get방식은 url 에 데이터를 포함시켜 요청 하는 것으로 보안유지가 불가능하다
따라서 로그인같은경우 get방식이이 부적합, 데이터를 header에 포함하여 천송하기 때문
header 에 데이터가 들어가기 때문에 전송하는ㄴ 길이에 제한이있어 초과데이터는 절단된다.

개인정보입력 -GET
